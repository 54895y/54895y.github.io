核心配置

config.yml 是 MatrixShop 的总入口。这里主要决定：

  • 数据库
  • 模块开关
  • 语言
  • 确认界面
  • 交易记录
  • 性能监控

完整示例​

database:
  type: sqlite
  table-prefix: matrixshop
  sqlite:
    file: data.db
  mysql:
    host: localhost
    port: 3306
    user: root
    password: ''
    database: matrixshop
    flags:
      - characterEncoding=utf-8
      - useSSL=false
      - allowPublicKeyRetrieval=true
      - useUnicode=true
      - connectionCollation=utf8mb4_unicode_ci

redis:
  enable: false
  host: localhost
  port: 6379
  user: ''
  password: ''
  connect: 32
  timeout: 1000
  reconnect-delay: 1000
  lock-prefix: matrixshop:lock

file:
  enable_listen: true

ui:
  confirm:
    enable: true
    default-amount: 1

recycle:
  enable: true

model:
  system-shop:
    enable: true
  player-shop:
    enable: true
  auction:
    enable: true
  global-market:
    enable: true
  chest-shop:
    enable: true
  player-trade:
    enable: true
  cart:
    enable: true
  mailbox:
    enable: true
  performance-monitor:
    enable: true
    warn-threshold-ms: 50
    bootstrap-warn-threshold-ms: 1000

transaction:
  enable: true
  retention-days: 30
  cleanup-interval-minutes: 60

language:
  default: zh_CN
  follow-client: false

global-market:
  distributed-lock:
    enable: true
    lease-ms: 15000
    acquire-timeout-ms: 3000
    retry-interval-ms: 50

1. database​

当前实现支持：

  • sqlite
  • mysql

常规单服推荐：

database:
  type: sqlite

如果你做跨服或较大体量市场，再切到 MySQL。

2. redis​

当前配置项已保留，但不要把它理解成“所有功能都已经走 Redis”。

它目前主要服务于分布式锁与后续扩展。
如果你没有明确的跨服需求，可以先保持关闭。

3. file.enable_listen​

file:
  enable_listen: true

开启后，插件会监听这些文件变化并尝试热重载：

  • config.yml
  • currency.yml
  • lang/**/*
  • ui/**/*
  • */module.yml
  • */shop/**/*
  • systemShop/goods/**/*
  • systemShop/recycle/**/*

兼容旧键：

file-listener: true

4. ui.confirm​

ui:
  confirm:
    enable: true
    default-amount: 1
  • enable：是否使用确认购买界面
  • default-amount：默认数量

兼容旧键：

  • confirm.enable
  • confirm.default-amount

5. model​

所有业务模块都在这里开关：

  • system-shop
  • player-shop
  • auction
  • global-market
  • chest-shop
  • player-trade
  • cart
  • mailbox

如果模块关闭：

  • 对应目录不会参与加载
  • 对应命令会被模块守卫拦截
  • 对应功能入口也不会工作

6. transaction​

transaction:
  enable: true
  retention-days: 30
  cleanup-interval-minutes: 60

这控制的是通用交易流水：

  • 系统商店购买
  • 玩家商店成交
  • 拍卖行成交
  • 全球市场成交
  • 箱子商店成交
  • 玩家交易中的货币流向

7. language​

language:
  default: zh_CN
  follow-client: false

推荐国服直接保持：

  • default: zh_CN
  • follow-client: false

这样所有玩家看到统一提示，不会因为客户端语言不同而切换。

8. global-market.distributed-lock​

这一段不只全球市场会用到，部分并发保护逻辑也会复用它的锁策略。

global-market:
  distributed-lock:
    enable: true
    lease-ms: 15000
    acquire-timeout-ms: 3000
    retry-interval-ms: 50

单服默认值可以直接保留。

9. 性能监控​

model:
  performance-monitor:
    enable: true
    warn-threshold-ms: 50
    bootstrap-warn-threshold-ms: 1000
  • warn-threshold-ms：常规任务告警阈值
  • bootstrap-warn-threshold-ms：启动阶段阈值

启动比平时更慢是正常的，所以这两个阈值分开配置了。

10. 最常见的几种改法​

只做系统商店​

model:
  system-shop:
    enable: true
  player-shop:
    enable: false
  auction:
    enable: false
  global-market:
    enable: false
  chest-shop:
    enable: false
  player-trade:
    enable: false
  cart:
    enable: true
  mailbox:
    enable: true

做玩家经济服​

model:
  player-shop:
    enable: true
  global-market:
    enable: true
  player-trade:
    enable: true
  mailbox:
    enable: true

做原版交互服​

model:
  chest-shop:
    enable: true
  player-trade:
    enable: true
  mailbox:
    enable: true
