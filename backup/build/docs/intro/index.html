MatrixShop 文档

这套文档对应当前仓库里的 MatrixShop 实现，不是旧版 Malkuth 文档的直接镜像。
文档目标只有两件事：

  • 让你按当前实现正确落地配置。
  • 明确哪些能力已经可用，哪些写法只是兼容旧配置。

插件定位​

MatrixShop 是一个基于 TabooLib 的模块化商店插件，当前已经包含这些主要模块：

  • 系统商店
  • 玩家商店
  • 拍卖行
  • 全球市场
  • 箱子商店
  • 玩家面对面交易
  • 购物车
  • 邮箱
  • 交易记录

兼容信息​

项目说明
运行兼容版本1.12.2+
推荐测试版本1.20.4 - 1.21.x
Java 版本Java 8+
推荐核心Paper、Purpur、Folia
混合核心Mohist、CatServer、Youer、Arclight、Magma、Banner
数据包前置TabooLib NMS、ProtocolLib、PacketEvents
数据库SQLite、MySQL、Redis
经济前置Vault、PlayerPoints、PlaceholderAPI

你会在文档里看到什么​

这套文档覆盖的是当前插件真正会读取、会生效的内容：

  • config.yml
  • currency.yml
  • 各模块目录下的 module.yml
  • systemShop/goods/*.yml
  • systemShop/shop/*.yml
  • systemShop/recycle/*.yml
  • 各模块的 ui/*.yml

同时也会说明：

  • 旧键名的兼容方式
  • 目录迁移规则
  • 外部物品库与头颅源的写法
  • 当前实现里不建议继续依赖的旧写法

使用建议​

如果你是第一次部署：

  1. 先看 快速开始
  2. 再看 核心配置
  3. 然后看 命令与绑定指令
  4. 最后按你的玩法选择模块页

如果你是在维护旧配置：

  1. 先看 模块目录结构
  2. 再看 兼容与最佳实践
  3. 然后看 常见问题与排错
  4. 最后逐模块迁移
