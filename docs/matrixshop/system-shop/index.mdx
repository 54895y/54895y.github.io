# 系统商店

系统商店是 MatrixShop 的核心功能之一，提供 NPC 商店、管理员商店等多种商店模式。

## 功能特性

- **多种商店模式** - 支持购买、出售、双向交易
- **商品配置** - 灵活的物品、价格、库存配置
- **回收规则** - 自动回收不需要的物品
- **权限控制** - 精细的权限管理

## 配置目录结构

```
plugins/MatrixShop/
├── systemShop/
│   ├── goods/          # 商品配置文件
│   ├── shop/           # 商店配置文件
│   └── recycle/        # 回收规则配置
```

## 快速开始

1. 在 `systemShop/goods/` 目录下创建商品配置文件
2. 在 `systemShop/shop/` 目录下创建商店配置文件
3. 使用命令将商店绑定到 NPC 或位置
