# MatrixShop 文档

MatrixShop 是一个功能强大的 Minecraft 商店插件，提供系统商店、玩家商店、拍卖行、全球市场等多种交易模式。

- **快速开始** - 5 分钟快速上手
- **核心配置** - 基础配置文件详解
- **货币配置** - 货币系统设置
- **模块目录结构** - 插件目录结构说明
- **命令与绑定** - 命令系统详解
- **占位符与价格显示** - 占位符使用指南

## 快速开始

1. 下载并安装 MatrixShop 插件
2. 将插件放入服务器的 plugins 目录
3. 启动服务器，生成配置文件
4. 修改配置文件，设置插件参数
5. 重启服务器，开始使用
