# MatrixShop 文档

MatrixShop 是一个面向生存、经济与市场玩法的模块化交易插件。当前主线版本已经完成统一命令体系、统一菜单框架、配置驱动绑定、直接业务 Kether 动作，以及 JDBC 优先的数据层实现。

## 当前版本入口

- [1.8.0 发布说明](/docs/matrixshop/release-notes-1-8-0)
- [更新日志总览](/docs/matrixshop/release-notes)

## 当前主线重点

- 绑定命令改为接近 TrMenu 的直接文本配置
- 菜单动作支持 `kether: matrixshop ...` 直接调业务
- `Bindings.Lang` 支持同级本地化覆盖
- `config.yml` 精简为 `debug / language / Scripts`
- 非豁免模块由 `shops/*.yml` 主导绑定与入口

## 当前实现模块

- `menu`
