# MatrixShop 开发者 API

MatrixShop 当前稳定公开层：

- `com.y54895.matrixshop.api`
- `com.y54895.matrixshop.api.model`
- `com.y54895.matrixshop.api.warehouse`

内部代码仍主要按这些分层组织：

- `MatrixShop`
- `core/command`
- `core/config`
- `core/database`
- `core/economy`
- `core/menu`
- `core/module`
- `core/warehouse`
- `module/*`

## 代码树

```text
src/main/kotlin/com/y54895/matrixshop/
```

## 生成 Javadoc

```bash
cd /www/codex-work/MatrixShop
bash ./gradlew generateJavadoc
```

输出目录：

```text
build/dokka/javadoc/
build/libs/MatrixShop-<version>-javadoc.jar
```
