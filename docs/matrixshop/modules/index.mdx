# 模块功能

本章节详细介绍 MatrixShop 的各个功能模块，包括玩家商店、拍卖行、全球市场等。

## 可用模块

- **系统商店** - NPC 商店、管理员商店配置
- **玩家商店** - 玩家创建和管理的个人商店
- **拍卖行** - 全服拍卖系统
- **全球市场** - 跨服交易功能
- **箱子商店** - 基于箱子的自动商店
- **玩家交易** - 玩家面对面交易
- **购物车/邮箱/交易记录** - 辅助功能模块

## 模块启用方式

在 `config.yml` 中启用或禁用对应模块：

```yaml
modules:
  player-shop:
    enabled: true
  auction:
    enabled: true
  global-market:
    enabled: false  # 跨服功能需要额外配置
```
