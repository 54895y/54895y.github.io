# 全球市场

全球市场使用 `mode: global_market`，适合全服统一浏览与购买玩家挂单。

## 目录

```text
plugins/MatrixShop/globalMarket/
├─ module.yml
├─ shop/
│  └─ globalMarket.yml
└─ ui/
   ├─ confirmGlobalMarket.yml
   ├─ globalMarketItem.yml
   ├─ globalMarketSort.yml
   └─ uploadGlobalMarket.yml
```

## 商店配置示例

```yml
id: globalMarket
mode: global_market
enable: true
title: '&6全球市场'
currency: vault
```

## 当前能力

- 全服公共上架
- 关键字过滤
- 排序切换
- 成交均价统计
- 交易记录写入
- 分布式锁保护并发成交

## 绑定命令

```text
open | upload | sort [type] | filter [keyword] | search [keyword]
```

## 排序与过滤

当前界面支持：

- 时间排序
- 价格排序
- 关键字筛选

当你切换排序或过滤条件后，浏览页会立即刷新。

## 成交说明

- 买家付款后，收益发给卖家
- 商品发给买家
- 成交写入交易记录
- 并发情况下会使用锁避免重复成交

## 适用场景

适合：

- 想做统一大市场
- 玩家不需要面对面交易
- 需要全服检索、排序、筛选

如果你更偏向本地交互或地皮式交易，优先考虑箱子商店。
