# 拍卖行

拍卖行使用 `mode: auction`，与普通系统商店共用 `ShopConfig`，但成交逻辑是拍卖模式。

## 目录

```text
plugins/MatrixShop/auction/
├─ module.yml
├─ shop/
│  └─ auction.yml
└─ ui/
   ├─ auctionItem.yml
   ├─ auctionMyBids.yml
   └─ auctionUpload.yml
```

## 商店配置示例

```yml
id: auction
mode: auction
enable: true
title: '&6拍卖行'
currency: vault
bid-increment: 10
auction-duration: 86400
```

## 关键字段

- `mode: auction`
- `currency`
- `bid-increment`
- `auction-duration`

## 绑定命令

拍卖行默认支持：

```text
open | upload | my_bids [page] | my_items [page]
```

## 成交流程

- 卖家上传主手物品并设置起拍价
- 买家出价时会校验余额
- 拍卖结束后自动结算
- 成交结果写入交易记录
- 物品与收益发放会尽量保证可回滚

## 推荐场景

适合需要：

- 抢拍
- 抬价
- 限时成交
- 和普通商店区分开的稀有交易区

如果你只想做固定一口价，不要用拍卖行，直接用全球市场或玩家商店。
