# 内置 UI 索引

下面是当前默认资源里常见的 UI 文件。

## 根目录共享 UI

| 文件 | 用途 |
| --- | --- |
| `ui/transaction.yml` | 通用交易记录菜单 |

## 系统商店

| 文件 | 用途 |
| --- | --- |
| `systemShop/ui/confirm.yml` | 普通系统商店确认菜单 |
| `systemShop/ui/favorite.yml` | 收藏夹菜单 |
| `systemShop/ui/order.yml` | 订单/排序相关保留菜单 |
| `systemShop/ui/priceCompare.yml` | 比价菜单 |
| `systemShop/ui/search.yml` | 搜索结果菜单 |

## 玩家商店

| 文件 | 用途 |
| --- | --- |
| `playerShop/ui/browsePlayerShops.yml` | 浏览挂单 |
| `playerShop/ui/confirmPlayerShop.yml` | 购买玩家挂单确认页 |
| `playerShop/ui/editGoods.yml` | 我的挂单管理 |
| `playerShop/ui/playerShopItem.yml` | 单个挂单模板 |
| `playerShop/ui/playerShopSettings.yml` | 卖家设置菜单 |
| `playerShop/ui/playerShopUnlock.yml` | 槽位解锁确认页 |
| `playerShop/ui/uploadShop.yml` | 上架页面 |

## 箱子商店

箱子商店的 UI 由模块内部默认菜单和对应模块资源共同驱动，默认资源会随着模块一起复制到 `chestShop/ui/`。
