# UI 系统

MatrixShop 提供灵活的 UI 界面配置系统，支持自定义 GUI 界面、交互动作等。

## UI 配置基础

所有 UI 配置文件位于 `plugins/MatrixShop/ui/` 目录下，使用 YAML 格式。

## 核心概念

- **模板（Templates）** - 预定义的界面布局
- **动作（Actions）** - 点击物品时执行的操作
- **占位符** - 动态显示信息
- **条件判断** - 根据条件显示不同内容

## 内置 UI

MatrixShop 提供多种内置 UI 模板：
- 商店界面
- 拍卖行界面
- 玩家交易界面
- 邮箱界面
- 购物车界面

## 自定义 UI

你可以基于内置模板创建自定义 UI，或完全从头设计新的界面。
