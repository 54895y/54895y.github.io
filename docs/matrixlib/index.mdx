# MatrixLib

`MatrixLib` 是 Matrix 系列插件的共享前置插件，当前统一承接：

- `branding / console / text / yaml / menu / compat / action / economy`
- 共享 `Economy/currency.yml` 货币定义与同步
- 共享 `bStats` API `MatrixBStats`
- MatrixAuth / MatrixCook / MatrixShop / MatrixStorage 的 telemetry 运行时依赖

## 当前发布

- 当前版本：`1.3.0`
