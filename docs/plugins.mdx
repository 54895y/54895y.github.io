# 插件列表

这里是 Matrix 系列插件的统一文档入口。

## MatrixAuth

混合登录与玩家档案管理插件，支持：

- SQLite / MySQL
- Folia
- Geyser / Floodgate 混合端
- PlaceholderAPI
- EasyBot 查询

[进入 MatrixAuth 文档](/docs/matrixauth/overview)

## MatrixShop

多功能商店插件，覆盖：

- 系统商店
- 玩家商店
- 拍卖行
- 全球市场
- 箱子商店

[进入 MatrixShop 文档](/docs/matrixshop/quick-start)
