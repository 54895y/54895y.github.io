# 插件列表

这里汇总当前站点已经维护的 Matrix 插件与相关技能文档。所有说明都以当前源码和默认资源文件为准，而不是旧版交接稿或历史目录结构。

## MatrixAgentSkills

MatrixAgentSkills 是 Matrix 系列插件的公开 skill、prompt pack 与知识库仓库，当前已经提供：

- `matrixshop-config`

适用场景：

- Codex 类 agent 原生安装 skill
- Claude / Gemini / Cline / Roo / Continue 导入 prompt
- AIChat 导入知识库后直接生成 MatrixShop 配置

[进入 MatrixAgentSkills 文档](/docs/matrix-agent-skills/overview)

如果你只是想直接复制一段提示发给 agent，请看：

- [直接发给 Agent 的示例](/docs/matrix-agent-skills/direct-link-examples)

## MatrixLib

MatrixLib 是 Matrix 系列的共享前置与运行时基础层，当前负责：

- 统一 `branding / console / text / yaml / menu / compat / action / economy`
- 共享货币配置与同步
- 共享 `bStats` API `MatrixBStats`
- 为 MatrixAuth / MatrixCook / MatrixShop / MatrixStorage 承接 shaded 遥测依赖

[进入 MatrixLib 文档](/docs/matrixlib)

## MatrixAuth

MatrixAuth 是混合登录与账号档案管理插件，围绕 `PREMIUM`、`OFFLINE`、`BEDROCK` 三种身份模式工作，支持：

- SQLite / MySQL 存储
- Geyser / Floodgate 基岩兼容
- PlaceholderAPI 扩展
- EasyBot 查询接口
- 玩家档案、多账号绑定与管理员强制维护

[进入 MatrixAuth 文档](/docs/matrixauth/overview)

## MatrixCook

MatrixCook 是可配置的烹饪系统插件，围绕锅具、配方、燃料和放置状态工作，支持：

- 自定义锅具、配方、燃料与分类
- 原版、CraftEngine、ItemsAdder 三种放置来源
- GUI 菜单、批量烹饪、燃料耗尽失败
- SQLite / MySQL / Redis / YAML 放置状态存储
- PlaceholderAPI、DecentHolograms、CMI 等可选集成

[进入 MatrixCook 文档](/docs/matrixcook/overview)

## MatrixShop

MatrixShop 是模块化交易插件。当前代码已经实现以下模块，并由统一命令和菜单框架驱动：

- Menu
- SystemShop
- PlayerShop
- GlobalMarket
- Auction
- ChestShop
- Transaction
- Cart
- Record

[进入 MatrixShop 文档](/docs/matrixshop)

## MatrixStorage

MatrixStorage 是 Matrix 系列的统一仓储插件，围绕邮箱、个人仓库、位置型实体仓库与外部仓库投递工作，当前代码已经提供：

- 玩家邮箱
- 个人仓库与待收区
- 仓库 / 邮箱扩容
- 实体仓库
- MatrixShop 外部仓库投递
- 原版方块、CraftEngine、ItemsAdder、Nexo 的方块 / 家具接入
- SQLite / MySQL / Redis / YAML 四后端

[进入 MatrixStorage 文档](/docs/matrixstorage)
