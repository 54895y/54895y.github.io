# MatrixStorage 开发者 API

MatrixStorage 当前没有单独拆分稳定 `api/` 包，开发者文档应优先参考这些分层：

- `MatrixStorage`
- `MatrixStoragePlatform`
- `bridge/`
- `integration/`
- `model/`
- `service/`
- `storage/`
- `ui/`

## 代码树

```text
src/main/kotlin/com/y54895/matrixstorage/
├─ MatrixStorage.kt
├─ MatrixStoragePlatform.kt
├─ bridge/
├─ command/
├─ integration/
├─ listener/
├─ metrics/
├─ model/
├─ service/
├─ storage/
└─ ui/
```

## 生成 Javadoc

```bash
cd /www/codex-work/MatrixStorage
bash ./gradlew generateJavadoc
```

输出目录：

```text
build/dokka/javadoc/
build/libs/MatrixStorage-<version>-javadoc.jar
```
