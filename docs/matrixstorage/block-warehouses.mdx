# 实体仓库

## 支持模式

- `terminal`
- `location`

`terminal` 仅作为个人仓库/邮箱入口。  
`location` 为真正的坐标仓库记录。

## 原版方块

当前原版方块仓库会优先复用方块自身原生库存。

这意味着：

- 原版容器会保留原生打开方式
- 漏斗搬运可通过事件开关进行限制
- 挖掉后内容会像原版容器一样掉落

## 自定义方块与家具

当前已接入：

- `CraftEngine`
- `ItemsAdder`
- `Nexo`

三类来源都支持：

- 放置识别
- 交互打开
- 破坏时移除仓库记录
- 虚拟库存掉落

## 设置项

当前设置页可修改：

- 显示名称
- 信任列表
- 漏斗插入开关
- 漏斗抽取开关
- 比较器输出开关
