# 仓库总览

`/warehouse` 与 `/matrixstorage` 默认都会进入总览中心。

总览页当前聚合：

- 个人仓库
- 待收区
- 邮箱入口
- 仓库扩容入口
- 邮箱扩容入口
- 玩家拥有的全部实体仓库列表

## 容量解锁

当前版本已经支持：

- 个人仓库格子解锁
- 邮箱格子解锁

每个解锁规则都支持：

- 多条件
- 多价格
- 多货币同时扣费

结算统一走 `MatrixLib` 的共享货币模块。

## 实体仓库列表

列表项当前展示：

- 显示名
- 仓库 ID
- 世界坐标
- 来源类型

点击行为：

- 左键打开仓库
- 右键打开设置
