# MatrixStorage

`MatrixStorage` 是 Matrix 系列里的统一仓储插件，当前实现围绕四条主线工作：

- 玩家邮箱
- 个人仓库与待收区
- 位置型实体仓库
- `MatrixShop` 外部仓库 SPI 对接

同时已经接入：

- 原版方块仓库
- `CraftEngine` 自定义方块与家具仓库
- `ItemsAdder` 自定义方块与家具仓库
- `Nexo` 自定义方块与家具仓库

当前代码基线：

- TabooLib 6
- Kotlin
- `MatrixLib` 作为硬前置插件
- `sqlite / mysql / redis / yaml` 四后端
